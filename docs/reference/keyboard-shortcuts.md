# Keyboard Shortcuts

All key bindings for the visualizer.

## Controls

| Key | Action | Details |
|-----|--------|---------|
| `Esc` | Quit | Closes the window |
| `Space` | Toggle audio source | Switches between Mic and Loopback |
| `F` | Toggle fullscreen | Borderless fullscreen on primary monitor |
| `R` | Randomize seed | Jumps camera to a new visual region |
| `D` | Toggle debug mode | Shows FPS + frame time in window title |
| `C` | Cycle color palette | 6 palettes: Electric Neon, Inferno, Deep Ocean, Vaporwave, Acid, Monochrome |
| `1`-`9` | Set sensitivity | `1` = 0.2x, `5` = 1.0x (default), `9` = 1.8x |
