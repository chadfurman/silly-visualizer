## Audio Inputs at a Glance

| Input | What drives it | Primary effect |
|-------|---------------|----------------|
| `bass` | Low freq (kick, bass) | Geometry scale, camera distance |
| `mids` | Mid freq (vocals, guitar) | Rotation speed, saturation |
| `highs` | High freq (cymbals, hats) | Glow, specular, carved detail |
| `energy` | Overall loudness | Fold complexity, FOV |
| `beat` | Rhythmic pulse | Color flash, chromatic aberration |
| `time` | Wall clock | Camera orbit, shape morph |
